# Dolibarr on Docker

Docker image for Dolibarr 17.0.0 with auto installer on first boot.

## What is Dolibarr ?

Dolibarr ERP & CRM is a modern software package to manage your organization's activity (contacts, suppliers, invoices, orders, stocks, agenda, ...).
